100% PURE FRONTEND

DASHBOARD


NĖURO is a client-only software that helps implement complex heterogeneous monitoring, control and analytical tools by using simple data models



Features:

  • Open source under the MIT License
  • No backend
  • No dependencies
  • Based on standard communication protocols and data models
  • Localization and Internationalization support
  • Desktop (Windows, macOS, Linux), Android and Browser / PWA clients are available
  • NĖURO is currently pre-alpha and moving rapidly toward its initial alpha release
  • Documentation and other materials on this site is still in early draft stage

Communication protocols:

  • WebSocket (push)
  • Server-sent Events (push)
  • HTTP Fetch (pull)
  • Local file read (pull)
